# awesome-sophnet-skills

Curated Sophnet skills bundle.

## Skills

All skills are documented here.

| Skill | Version | Function |
| --- | --- | --- |
| `sophnet-face-search` | `v1.0.0` | Face detection + similarity search across images via Sophnet API. |
| `sophnet-auto-skill-dispatcher` | `v1.0.0` | README-first matching that discovers and installs required Sophnet skills for the current task. |
| `sophnet-image-generate` | `v1.0.0` | Text-to-image generation via Sophnet (model selection + stable polling outputs). |
| `sophnet-image-ocr` | `v1.0.0` | OCR text/table extraction from images (local files or URLs) via Sophnet OCR API. |
| `sophnet-skill-installer` | `v1.0.0` | Lists/installs Sophnet skills (defaulting to this repo). |
| `sophnet-smart-image-loader` | `v1.0.0` | Resolves an image URL or local path into a local file suitable for preview tools. |
| `sophnet-sophon-key` | `v1.0.0` | Resolves/persists `SOPH_API_KEY` from env or Moltbot config; guides setup when missing. |
